🤖 COSMOpet Virtual Assistant – Simulation Preview

This is a visual simulation of COSMOpet’s chatbot interface, designed for future implementation in collaboration with developers.
The assistant will guide children and caregivers through:

  • 📬 Contact with nutritionist, psychologist, music therapist, or local coordinator
  • 📍 Personalized content based on geolocation (via Galileo GNSS)
  • 🎯 Triggered content based on Copernicus environmental stressors
  • 📨 Email request system with consent-based data collection (GDPR-compliant)

The chatbot will activate different responses depending on user selection. All sensitive data is simulated in this demo.

🔄 Prototype logic only – Interaction features under development for pilot phase 2025-2026.
